# Chess House Landing Page

This is the official "Coming Soon" landing page for Chess House.

## Branding
- **Primary Color:** Royal Navy (#002366)
- **Secondary Color:** Gold (#D4AF37)
- **Typography:** Playfair Display (Headings), Inter (Body)

## GitHub Pages Deployment
To publish this page to GitHub Pages:
1. Push this `final-chesshouse` directory to a new GitHub repository.
2. Go to **Settings > Pages** in your repository.
3. Select the branch (usually `main`) and the folder (root) to deploy from.
4. Your site will be live at `https://<your-username>.github.io/<repo-name>/`.

## Files
- `index.html`: The main landing page.
- `styles.css`: The styling for the page.
- `favicon.svg`: The site's favicon.
